# Expl Compiler

This repository contains a compiler for Expl, a programming language designed for educational purposes. The compiler translates Expl source code into assembly code compatible with the XSM simulator.

## Features

- Lexical Analysis
- Syntax Analysis
- Semantic Analysis
- Code Generation
